# 🍕 Alfredo Restaurant Website

This is a simple HTML & CSS project simulating a pizza restaurant. It includes a homepage, menu table, store hours, and a contact form.

## 📂 Pages

- `index.html` - Home page
- `storehours.html` - Store hours
- `contact.html` - Contact form

## 🎨 Tech Used

- HTML5
- CSS3
